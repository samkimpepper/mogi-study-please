# Memory

이 레포는 모기(사용자)의 백엔드 이직 준비용 공부 레포지토리다.

## 기록 방식

- `TIL.md`는 매일 공부한 주제를 날짜별로 한 줄 체크리스트 형태로만 기록한다.
- 사용자가 붙여넣은 문단이나 원문 요약을 `TIL.md`에 그대로 넣지 않는다.
- 공부한 심층 내용은 필요할 때 별도 디렉토리와 별도 Markdown 파일로 정리한다.
- 단, 심층 내용도 사용자가 명시적으로 "기록해줘"라고 요청했을 때만 파일에 저장한다.
- 사용자가 문단을 붙여넣으면 우선 대화로 같이 이해하고 정리한다.
- 파일 기록은 사용자가 깨달은 점, 나중에 다시 보고 싶은 점, 또는 직접 기록해달라고 한 내용 위주로 한다.

## 사용자 맥락

- 사용자는 주니어 백엔드 개발자이며 백엔드 이직을 준비하고 있다.
- 설명은 백엔드 개발자 관점에서 실무적인 비유와 구조 중심으로 해주면 좋다.
- 사용자는 가볍고 편한 말투를 쓰지만, 내용은 제대로 이해하고 싶어 한다.
