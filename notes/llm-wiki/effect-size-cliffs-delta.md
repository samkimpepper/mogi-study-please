# Effect Size와 Cliff's Delta 기초

> [!summary]
> `effect size`는 차이가 있는지보다 차이가 얼마나 큰지를 보는 값이다.
>
> p-value가 "이 차이를 믿을 만한가?"에 가깝다면, effect size는 "이 차이가 실무에서 신경 쓸 만큼 큰가?"에 가깝다.

## 왜 봐야 하나

LLM wiki 검색 평가에서 p-value만 보면 판단이 부족할 수 있다.

예를 들어 다음 결과가 나왔다고 하자.

```text
old 평균 점수: 0.72
new 평균 점수: 0.74
p-value: 0.01
```

p-value만 보면 통계적으로 유의해 보인다.

하지만 실제 차이는 `0.02`뿐이다.

이때 effect size 관점에서는 다음을 묻는다.

```text
그 0.02 차이가 운영에서 의미 있나?
사용자가 체감할 만큼 큰가?
비용, 지연시간, 복잡도 증가를 감수할 만큼 큰가?
```

> [!info]
> 백엔드 관점에서 보면 "응답 시간이 통계적으로 2ms 줄었다"와 비슷하다.
>
> 트래픽이 아주 크거나 SLA가 빡빡하면 의미 있을 수 있지만, 일반적인 기능에서는 배포 판단을 바꿀 만큼 중요하지 않을 수도 있다.

## p-value와 Effect Size 차이

`p-value`는 귀무가설이 맞다고 가정했을 때, 지금 관측한 결과만큼 이상한 결과가 나올 가능성을 보는 값이다.

반면 `effect size`는 관측된 차이의 크기를 본다.

```text
p-value
= 이 차이가 우연이라고 보기 어려운가?

effect size
= 이 차이가 얼마나 큰가?
= 실무적으로 의미 있는 크기인가?
```

그래서 실험 결과를 볼 때는 둘을 같이 봐야 한다.

| p-value | effect size | 해석 |
| --- | --- | --- |
| 작음 | 큼 | 꽤 믿을 만하고, 실무적으로도 의미 있을 가능성이 큼 |
| 작음 | 작음 | 차이는 있어 보이지만, 실무적으로는 별 의미 없을 수 있음 |
| 큼 | 큼 | 좋아 보이지만 데이터가 부족할 수 있음. 추가 실험 후보 |
| 큼 | 작음 | 굳이 바꿀 근거가 약함 |

> [!warning]
> 통계적으로 유의하다는 말은 실무적으로 중요하다는 뜻이 아니다.
>
> 표본이 아주 많으면 매우 작은 차이도 p-value가 작게 나올 수 있다.

## p-value는 애매하지만 효과는 커 보이는 경우

반대로 이런 결과도 있을 수 있다.

```text
old 평균 점수: 0.62
new 평균 점수: 0.75
p-value: 0.08
```

보통 기준을 `0.05`로 잡았다면 통계적으로 유의하다고 말하기 어렵다.

하지만 차이는 꽤 커 보인다.

이때는 다음처럼 해석할 수 있다.

```text
이번 데이터만으로 확신하기는 어렵다.
다만 효과 크기는 커 보인다.
추가 실험을 해볼 가치가 있다.
```

> [!tip]
> p-value가 크다고 바로 "효과 없음"이라고 말하면 안 된다.
>
> 표본이 적어서 효과를 충분히 증명하지 못했을 수도 있다.

## Cliff's Delta

`Cliff's delta`는 두 그룹을 비교했을 때, 한쪽 값이 다른 쪽보다 얼마나 자주 큰지를 보는 effect size 지표다.

비모수 검정처럼 정규분포 가정을 강하게 두기 어려운 상황과 잘 어울린다.

LLM wiki 검색 평가에서는 다음 질문에 가깝다.

```text
new 검색기 점수가 old 검색기 점수보다 더 큰 경우가 얼마나 자주 나오나?
```

값의 범위는 대략 다음과 같다.

```text
+1  : new가 거의 항상 old보다 큼
 0  : 둘이 비슷함
-1  : new가 거의 항상 old보다 작음
```

직관적으로는 다음처럼 볼 수 있다.

```text
Cliff's delta = new가 이긴 비율 - old가 이긴 비율
```

예를 들어 모든 비교쌍 중에서 결과가 다음과 같다고 하자.

```text
new가 old보다 큼: 70%
old가 new보다 큼: 20%
동점: 10%
```

그러면 Cliff's delta는 다음과 같다.

```text
0.70 - 0.20 = 0.50
```

이 값은 new 쪽으로 꽤 기울어져 있다는 뜻이다.

> [!info]
> Cliff's delta는 평균 차이를 직접 보는 것보다 "한쪽이 다른 쪽보다 자주 이기는가?"에 더 가깝다.

## 검색 평가에서 읽는 법

LLM wiki 검색 PR에서 다음처럼 나온다고 하자.

```text
Wilcoxon signed-rank test: p = 0.04
Cliff's delta: 0.12
```

이 경우는 이렇게 읽을 수 있다.

```text
통계적으로 차이가 있다고 볼 근거는 있다.
하지만 효과 크기는 작아 보인다.
실무적으로 의미 있는 개선인지는 추가로 봐야 한다.
```

반대로 다음처럼 나올 수도 있다.

```text
Wilcoxon signed-rank test: p = 0.09
Cliff's delta: 0.55
```

이 경우는 이렇게 읽을 수 있다.

```text
이번 데이터만으로 통계적으로 확정하기는 어렵다.
하지만 new가 old보다 꽤 자주 이기는 흐름은 있어 보인다.
gold task를 늘려서 다시 확인해볼 가치가 있다.
```

## 한 번에 정리

| 개념 | 뜻 | 기억할 점 |
| --- | --- | --- |
| p-value | 이 차이가 우연이라고 보기 어려운지 보는 값 | 믿을 만한가에 가까움 |
| effect size | 차이가 얼마나 큰지 보는 값 | 실무적으로 중요한가에 가까움 |
| Cliff's delta | 한쪽 값이 다른 쪽보다 얼마나 자주 큰지 보는 지표 | 비모수적인 비교와 잘 어울림 |
| 통계적 유의성 | 차이가 있다고 볼 근거가 충분한지 판단 | 실무적 중요성과 다름 |

## 나중에 다시 볼 포인트

- p-value는 "믿어도 되나?"에 가깝다.
- effect size는 "차이가 얼마나 크나?"에 가깝다.
- 통계적으로 유의해도 효과 크기가 작으면 실무적으로 별 의미 없을 수 있다.
- p-value가 유의하지 않아도 효과 크기가 크면 추가 실험 후보가 될 수 있다.
- Cliff's delta는 new가 old보다 얼마나 자주 이기는지를 보는 effect size 지표다.
